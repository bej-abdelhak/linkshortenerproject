# Agent Instructions

This repository maintains coding standards and LLM agent guidelines in the `docs/` folder.

Before generating any code, agents MUST read the relevant individual instruction files in `docs/` for the task they are performing. Treat this as a required first step, not a suggestion.

**See: [docs/AGENTS.md](docs/AGENTS.md)**

For any updates to agent behavior, Clerk rules, or project conventions, please edit `docs/AGENTS.md`.
